Privacy-first, end-to-end secure web application

Overview

VaultX allows users to store, manage, and retrieve sensitive information and attachments safely. Built with modern authentication, AES-256-GCM encryption, and short-lived pre-signed URLs, VaultX ensures only the rightful owner can access their secrets and files.

Key Features

  • User Authentication & Account Management: Secure sign-up, login, password hashing (bcrypt), and reset password with hashed tokens.
  • Secrets Management: Create, view, delete, and manage personal secrets with full privacy controls.
  • File Attachments: Upload up to 5 files per secret, stored encrypted with short-lived download links.
  • Security: AES-256-GCM encryption, token-based authentication, strict user data isolation.

Tech Stack

  • Frontend: HTML, CSS, JavaScript (Vanilla)
  • Backend: Node.js, Express.js
  • Database: MongoDB
  • Authentication: JWT, bcrypt
  • Encryption: AES-256-GCM
  • File Storage: Amazon S3 with pre-signed URLs

Future Enhancements

  • Two-Factor Authentication (2FA)
  • Rate Limiting & Bruteforce Protection
  • Activity Logs & Audit Trails
  • Secret Sharing with Expiry
  • Optional Client-Side Encryption
